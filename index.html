Installation

To start using Slize.js directly, first download the slize.js file by clicking the download button at the top of this page. Then add the following script to the head of your HTML document: 

<script src="dist/slize.js"></script>

If you want to use slize.js cdn, copy the following link. Then add it to the head of your HTML document using the script tag: 

<script src="https://cdn.jsdelivr.net/gh/ubeyin/slize/slize.js"></script>

Copy

You can use slize.js with version. The latest version of slize.js is 1.0.

Documentation

For use unique (one) function, use the following snippet. The following snippet is recommended for you: 

var sl = new Slize();
sl.go(function($){
  // Your code goes here
});

Here, The new Slize() is used to access slize and $ is used to call slize functions. You can give something else instead of the dollar ($) sign.

For use global function, use the following snippet: 

sl(...).function(...);
// Examples
sl("https://www.sololearn.com").share("Sololearn", "Sololearn: Learn codes for free");

Here, The sl is used for access slize and all functions.

# Scope

The scope is the binding part between the HTML for view and the JavaScript for control (like angular js).

  • View, set data in HTML for view.
  • Data, set data from XHR.
  • Control, makes/changes/controls the data with JS.
$("body").scope({
  heading: "This is a heading text"
});

In HTML file, 

<body>
  <h1> {{heading}} ! </h1>
</body>


# Slideshow

The scope is the binding part between the HTML for view and the JavaScript for control (like angular js).

var slide = $("img").slide(1000);
// Switch to next or previous
slide.next();
slide.prev();

Here, '1000' is the duration of slideshow (milliseconds).

In HTML file, 

<div class="carousel">
  <img src="flowers1.jpg"/>
  <img src="flowers2.jpg"/>
  <img src="flowers3.jpg"/>
 </div>
