Installation

To start using Slize.js directly, first download the slize.js file by clicking the download button at the top of this page. Then add the following script to the head of your HTML document: 

<script src="dist/slize.js"></script>

If you want to use slize.js cdn, copy the following link. Then add it to the head of your HTML document using the script tag: 

<script src="https://cdn.jsdelivr.net/gh/ubeyin/slize/slize.js"></script>

Documentation

For use unique (one) function, use the following snippet. The following snippet is recommended for you: 

var sl = new Slize();
sl.go(function($){
// Your code goes here
});

For use global function, use the following snippet: 

sl(...).function(...);
